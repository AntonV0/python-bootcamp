## Short Answer Test

### Q1.  
Explain why using `virtual environments` is a critical best practice in software development.

**Answer:**  
Virtual environments isolate project dependencies so different projects can use different package versions without conflicts. This ensures consistency across machines and makes projects easier to reproduce and maintain.

---

### Q2.  
What is the difference between an interpreter and an IDE?

**Answer:**  
An interpreter executes code, while an IDE is a development tool that helps write, edit, and debug code. An IDE relies on an interpreter to run programs.

---

### Q3.  
Describe the roles of a linter and a formatter. Can one replace the other? Why or why not?

**Answer:**  
A linter identifies errors, potential bugs, and poor coding practices, while a formatter automatically enforces consistent code style. They cannot replace each other because linters analyse how correct the code is, while formatters focus on the appearance and consistency of the code.
