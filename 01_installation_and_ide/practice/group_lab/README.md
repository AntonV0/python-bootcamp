This project demonstrates how Python teams share and recreate development environments using a `requirements.txt` file.

## Overview
- Dependencies are captured using `pip freeze`
- A second developer recreates the environment using `pip install -r requirements.txt`
- This workflow helps ensure consistent environments across a team

## Tools Used
- Python
- Virtual Environments (`venv`)
- pip
- Git & GitHub
